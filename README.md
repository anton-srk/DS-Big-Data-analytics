# Big Data Analytics

This repo contains a bunch of homeworks completed by me during the Big Data Analytics course (https://dataschool.digital/). These are organized as follows:

* 01-12: Basic ML algorithms studied using some toy examples (some neat XGBoost parameter tuning in 09, clustering with the help of T-sne in 11);  
* 13: A nice exercise in NLP using the IMDB review dataset (Word2Vec);  
* 14-15: Introduction to neural networks;  
* 16: Intro to recurrent neural networks, includes some interesting code with Word2Vec embeddings;   
* 17: A basic movie recommender system example;
* 18: An exercise in social network analysis (Link prediction);
* 19: Final project, "1C Sales prediction" on kaggle (https://www.kaggle.com/c/competitive-data-science-predict-future-sales)
