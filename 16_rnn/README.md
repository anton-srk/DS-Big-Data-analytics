# HW 16: RNN

* Solve the sentiment classification problem on the IMDB review dataset using LSTM and GRU RNN. Study the influence of the NN parameters on the classification score. Try preliminary embedding of words using Word2Vec.  

The code is organized as follows:

_hw16_lstm_ contains the study of LSTM models (_hw16_gru_ is the same, but in case of GRU)  

_hw16_word2vec_lstm_16features_ is the LSTM model with preliminary Word2Vec embedding (similar files are using different word2vec vector dimensionalities)  

_hw16_word2vec_lstm_plotting__ contains the graphs of all the LSTM models using Word2Vec embedding.
